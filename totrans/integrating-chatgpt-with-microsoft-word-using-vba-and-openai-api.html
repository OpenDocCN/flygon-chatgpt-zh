Chapter 1: Introduction to VBA and OpenAI API

 

In this chapter, we’ll explore the basics of VBA and the OpenAI API, setting the stage for integrating these two powerful tools.

What is VBA?

Visual Basic for Applications (VBA) is a programming language developed by Microsoft. It is primarily used for automating repetitive tasks in Microsoft Office applications such as Word, Excel, and Outlook. With VBA, users can write custom scripts (macros) to perform a wide range of tasks, from simple formatting to complex data manipulation.

• Common Uses of VBA in Microsoft Word:
– Automating repetitive text formatting tasks
– Creating custom dialog boxes and forms
– Generating reports and documents automatically
– Integrating with other Microsoft Office applications

What is the OpenAI API?

The OpenAI API is a cloud-based service that provides access to various AI models, including ChatGPT. These models can understand and generate human-like text based on the prompts provided. By integrating the OpenAI API with VBA, you can automate the interaction with these models directly from within Microsoft Word.

• How the OpenAI API Enables AI Interactions:
– The API allows you to send text-based prompts to the AI model.
– The AI processes these prompts and returns responses that can be used within your applications.
– This interaction can be customized and automated using VBA, making it a powerful tool for enhancing productivity in Word.

Why Integrate VBA with OpenAI?

Integrating VBA with the OpenAI API allows you to create a more dynamic and interactive experience within Microsoft Word. Here are a few benefits:

• Enhanced Document Creation: Automatically generate or suggest content based on AI responses.
• Automated Responses: Use AI to respond to repetitive queries or document formatting tasks.

Opening the VBA Editor

The VBA Editor is where you write and manage your VBA code. Here’s how to access it:

Chapter 4: Creating Your First VBA Macro

 

With a basic understanding of the VBA editor, you can now create your first practical VBA macro that interacts with Word documents.

Writing a Simple Macro

In this section, we’ll create a macro that automates a basic text manipulation task in Word.

• Example: Formatting Selected Text:
 vba
 Sub BoldSelectedText()
  Selection.Font.Bold = wdToggle
 End Sub
– Explanation: This macro toggles bold formatting on the currently selected text.

Debugging Your Macro

Writing code without errors is challenging, but the VBA editor provides tools to help you debug.

• Using the Immediate Window:
– The Immediate Window lets you test single lines of code.
– Example: Select or write some text in the current Word document and select it. Next, type ?Selection.Text in the Immediate Window and press Enter to display the currently selected text in the Immediate Window.
• Setting Breakpoints:
– Click in the left margin of the code window to set a breakpoint.
– Breakpoints pause code execution, allowing you to inspect variables and step through your code line by line.

Enhancing Your Macro

You can improve your macro by adding features like error handling or more complex formatting options.

• Example: Adding Error Handling:
  
  
 vba
 Sub BoldSelectedText()
  On Error GoTo ErrorHandler
  Selection.Font.Bold = wdToggle
  Exit Sub
 ErrorHandler:
  MsgBox "An error occurred: " & Err.Description
 End Sub
– Explanation: This code includes an error handler: On Error GoTo ErrorHandler. Any line of code below that that causes an error will cause processing to jump to the line below ErrorHandler: and display a message box if something goes wrong.

 

Chapter 5: Understanding the OpenAI API

 

To interact with ChatGPT, it’s essential to understand how the OpenAI API works. This chapter will cover the basics of making API requests and the structure of the API responses.

API Basics

An API (Application Programming Interface) allows different software applications to communicate with each other. The OpenAI API provides a way to interact with AI models like ChatGPT.

• How the API Works:
– You send a request to the API with specific parameters.
– The API processes your request and returns a response containing the AI's output.

API Endpoints

API endpoints are specific URLs where you can send requests to interact with different features of the API.

• Endpoint for ChatGPT:
– https://api.openai.com/v1/chat/completions is the primary endpoint for interacting with ChatGPT models.

API Parameters

When making a request, you’ll include parameters that control how the API behaves.

• Key Parameters for ChatGPT:
– model: Specifies which model to use (e.g., "gpt-4o").
– messages: The conversation history, including system and user messages.
– temperature: Controls the creativity of the response (lower values make responses more focused).

Chapter 7: Making API Calls from VBA (with JSONConverter)

 

In this chapter, you’ll learn how to make API calls from VBA using the JSONConverter tool, allowing you to interact with the OpenAI API.

Installing the JSONConverter Library

To handle JSON data within VBA, you'll need to install the JSONConverter tool from the VBA-JSON GitHub repository.

• Steps to Install JSONConverter:
– Download the JsonConverter.bas file from the author’s VBA-JSON GitHub repository. This is a Windows-only supported version that requires you to include a reference to the "Microsoft Scripting Runtime" in the VBA editor.
– Open the VBA editor in Word.
– Go to File > Import File and select the JsonConverter.bas file.
– Once imported, the JSONConverter module will be available for use in your project. If you get the following error when running the example code in this chapter, go to Tools > References… in the VBA editor and make sure “Microsoft Scripting Runtime” is selected. If not, select it from the list (scroll down if you have to) then click [OK].

Setting Up an HTTP Request in VBA

Now that you have the JSONConverter installed, you can use it to send and receive JSON data when interacting with the OpenAI API.

• Writing VBA Code for an API Request:
– Use MSXML2.ServerXMLHTTP or WinHttp.WinHttpRequest.5.1 to send HTTP requests.
– Example code:
 vba
 Dim http As Object
 Set http = CreateObject("MSXML2.ServerXMLHTTP")
  
 http.Open "POST", "https://api.openai.com/v1/chat/completions", False
 http.setRequestHeader "Content-Type", "application/json"
 http.setRequestHeader "Authorization", "Bearer YOUR_API_KEY_HERE"
• Creating the JSON Payload:
– Use JSONConverter to create the JSON object for the request body.
 vba
 Dim json As Object
 Set json = JsonConverter.ParseJson("{}")
  
 json("model") = "gpt-4o"
 json("messages") = Array( _
  JsonConverter.ParseJson("{""role"": ""system"", ""content"": ""You are a helpful assistant.""}"), _
  JsonConverter.ParseJson("{""role"": ""user"", ""content"": ""Hello, how can you help me learn Word VBA?""}") _
 )
  
 Dim jsonString As String
 jsonString = JsonConverter.ConvertToJson(json)
• Sending the Request and Handling the Response:
– Send the JSON payload to the OpenAI API and capture the response.
 vba
 http.send jsonString
 Dim response As String
 response = http.responseText

Summing Up

In Chapters 7 and 8, we've covered how to install the JSONConverter library, set up an HTTP request in VBA, and handle API responses from the OpenAI API. Now, let's combine everything into a complete, working example that you can use in your own projects.

• Working Example: Sending a Request to the OpenAI API

This example demonstrates how to send a text prompt to ChatGPT using VBA, retrieve the response, and display it in a message box. This is a practical example you can build upon as you develop more complex integrations.

Step 1: Install the JSONConverter Library

Ensure that you've downloaded and installed the JsonConverter.bas file from the author’s VBA-JSON GitHub repository.

  • In the VBA editor, go to File > Import File and select the JsonConverter.bas file.
  • This will add the JSONConverter module to your project, which will allow you to work with JSON data in VBA.

Step 2: Define the Function to Interact with the API

Create a new module in the VBA editor and paste the following code. This code sets up an HTTP request to the OpenAI API, sends a prompt, and retrieves the AI's response.

 vba
 ' Example of interacting with the OpenAI API using VBA
 Sub GetChatGPTResponse()
  ' Define the API endpoint
  Dim apiUrl As String
  apiUrl = "https://api.openai.com/v1/chat/completions"
 
  ' Retrieve the API key from an environment variable
  Dim apiKey As String
  apiKey = Environ("OPENAI_API_KEY")
 
  ' Set up the HTTP request
  Dim http As Object
  Set http = CreateObject("MSXML2.ServerXMLHTTP")
 
  ' Open the request
  http.Open "POST", apiUrl, False
 
  ' Set the request headers
  http.setRequestHeader "Content-Type", "application/json"
  http.setRequestHeader "Authorization", "Bearer " & apiKey
 
  ' Prepare the JSON payload
  Dim json As Object
  Set json = JsonConverter.ParseJson("{}")
 
  ' Set the model and the messages
  json("model") = "gpt-4o"
  json("messages") = Array( _
  JsonConverter.ParseJson("{""role"": ""system"", ""content"": ""You are a helpful assistant.""}"), _
  JsonConverter.ParseJson("{""role"": ""user"", ""content"": ""Hello, how can you help me learn VBA?""}") _
  )
 
  ' Convert the JSON object to a string
  Dim jsonString As String
  jsonString = JsonConverter.ConvertToJson(json)
 
  ' Send the request
  http.send jsonString
 
  ' Get the response
  Dim response As String
  response = http.responseText
 
  ' Parse the JSON response
  Dim jsonResponse As Object
  Set jsonResponse = JsonConverter.ParseJson(response)
 
  ' Extract the AI's response
  Dim chatResponse As String
  chatResponse = jsonResponse("choices")(1)("message")("content")
 
  ' Display the response in a message box
  MsgBox "ChatGPT says: " & chatResponse
 End Sub

 Function SendToChatGPT(userInput As String) As String
  Dim http As Object
  Set http = CreateObject("MSXML2.ServerXMLHTTP")

  json("model") = "gpt-4o"

Handling Multiple Messages

To create a dynamic conversation with ChatGPT, it’s important to manage the history of the conversation effectively. This section will guide you on how to modify your existing code to handle multiple messages in a conversation.

Step 1: Adding a Collection to Store Messages

To keep track of the conversation, you’ll need a collection that stores all messages exchanged between the user and the assistant. This collection will be included in the SendToChatGPT function, which you created in Chapter 8.

  1. Declare the Collection at the Module Level:

You should declare the collection at the module level to ensure that it persists between different subroutine and function calls.

 vba
 ' Declare the message history collection at the top of your module
 Private messageHistory As Collection
  1. Initialize the Collection in Your Function:

Modify the SendToChatGPT function to initialize the messageHistory collection if it hasn’t been initialized already. This ensures that the collection is available for storing and retrieving messages throughout the session.

 vba
 Function SendToChatGPT(userInput As String) As String
  Dim http As Object
  Set http = CreateObject("MSXML2.ServerXMLHTTP")
 
  ' Retrieve the API key from an environment variable
  Dim apiKey As String
  apiKey = Environ("OPENAI_API_KEY")
 
  ' API request setup
  http.Open "POST", "https://api.openai.com/v1/chat/completions", False
  http.setRequestHeader "Content-Type", "application/json"
  http.setRequestHeader "Authorization", "Bearer " & apiKey
 
  ' Initialize the message history collection if it doesn't exist
  If messageHistory Is Nothing Then
  Set messageHistory = New Collection
  ' Add the initial system message to the collection
  messageHistory.Add JsonConverter.ParseJson("{""role"": ""system"", ""content"": ""You are a helpful assistant.""}")
  End If

Step 2: Adding Messages to the Collection

Next, you’ll add the user’s input and the assistant’s responses to the messageHistory collection.

  1. Add the User’s Input:

After collecting the user’s input (which is done via the TextBox on your UserForm), add it to the messageHistory collection.

 vba
 ' Adding the user message
 messageHistory.Add JsonConverter.ParseJson("{""role"": ""user"", ""content"": """ & userInput & """}")
  1. Prepare the JSON Payload Using the Message History:

Use the messageHistory collection to create the JSON payload for the API call.

 vba
 ' Prepare the JSON payload with the full conversation history
 Dim json As Object
 Set json = JsonConverter.ParseJson("{}")
 json("model") = "gpt-4o"
  
 ' Convert the message history to an array and assign it to the 'messages' key
 Dim messageArray() As Variant
 ReDim messageArray(0 To messageHistory.Count - 1)
  
 Dim i As Integer
 For i = 1 To messageHistory.Count
  Set messageArray(i - 1) = messageHistory(i)
 Next i
  
 ' Convert the message array to JSON for the API call
 json("messages") = messageArray

Step 3: Handling the AI’s Response

After sending the API request and receiving the AI’s response, add the AI’s reply to the messageHistory collection.

  1. Add the AI’s Response to the History:

Extract the AI’s response from the JSON response and add it to the messageHistory collection.

 vba
 ' Convert the JSON object to a string
 Dim jsonString As String
 jsonString = JsonConverter.ConvertToJson(json)
 
 ' Send the request
 http.send jsonString
 
 ' Get the response
 Dim response As String
 response = http.responseText
 
 ' Parse the JSON response
 Dim jsonResponse As Object
 Set jsonResponse = JsonConverter.ParseJson(response)
 
 ' Extract the AI's response
 Dim chatResponse As String
 chatResponse = jsonResponse("choices")(1)("message")("content")
  
 ' Add the AI's response to the collection
 messageHistory.Add JsonConverter.ParseJson("{""role"": ""assistant"", ""content"": """ & chatResponse & """}")
  
 SendToChatGPT = chatResponse
 End Function

This complete function is now fully equipped to handle multi-turn conversations, where the assistant retains context and can engage in more meaningful interactions with the user.

  ' Add comment with the explanation

 Function GetClauseText(ByVal pattern As String) As Variant
  
  InitializeStandardClauses
  Dim rng As Range
  Set rng = ActiveDocument.Content
  With rng.Find
  .Text = pattern
  .MatchWildcards = True
  Do While .Execute(FindText:=pattern, Forward:=True) = True
  GetClauseText = Array(StandardClauses(rng.Text), rng)
  Loop
  End With
 End Function
  
 Function GetImprovementSuggestion(ByVal clause As String, ByVal clauseText As String) As String
  ' Example using the OpenAI API to generate improvement suggestions
  Dim json As Object
  Dim http As Object
  Set http = CreateObject("MSXML2.ServerXMLHTTP")
 
  http.Open "POST", "https://api.openai.com/v1/chat/completions", False
  http.setRequestHeader "Content-Type", "application/json"
  http.setRequestHeader "Authorization", "Bearer " & Environ("OPENAI_API_KEY")
 
  Set json = JsonConverter.ParseJson("{}")
  json("model") = "gpt-4o"
  json("messages") = Array( _
  JsonConverter.ParseJson("{""role"": ""system"", ""content"": ""You are a legal assistant.""}"), _
  JsonConverter.ParseJson("{""role"": ""user"", ""content"": ""Here is a clause: " & clauseText & " Suggest improvements or alternatives based on the standard clause: " & StandardClauses(clause) & """}") _
  )
 
  Dim jsonString As String
  jsonString = JsonConverter.ConvertToJson(json)
 
  http.send jsonString
  Dim response As String
  response = http.responseText
 
  Dim jsonResponse As Object
  Set jsonResponse = JsonConverter.ParseJson(response)
 
  GetImprovementSuggestion = jsonResponse("choices")(1)("message")("content")
 End Function
