# 自然语言处理（NLP）

![image](img/chapter_title_below.png)

NLP 是人工智能（AI）领域的一个研究方向，它专注于计算机和人类语言之间的交互。

它使计算机能够以有意义的方式理解、解释和生成人类语言。ChatGPT 利用 NLP 来理解和回应用户输入。

例如，如果你问 ChatGPT，“今天天气如何？”它会使用 NLP 来识别你的问题并生成一个合适的回答，比如“今天天气晴朗温暖”。
