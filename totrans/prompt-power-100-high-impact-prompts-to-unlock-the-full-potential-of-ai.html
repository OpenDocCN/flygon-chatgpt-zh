How to Use the Prompts (Step-by-Step)

1.

Copy and Paste the Prompt

Each prompt in this book is plug-and-play. Simply copy the text and paste it into your AI chat box.

 

2.

Edit the [Variables]

Many prompts contain brackets like:

[your niche] — [target audience] — [product name]

Replace those parts with your specific details.

This makes the AI generate results customized to you.

3.

Ask Follow-Up Questions

AI loves context. If the first result isn’t perfect, reply with:

“Make it shorter”

“Rewrite in a fun tone”

“Add bullet points”

“Now translate to Portuguese”

“Turn this into a tweet”

Think of it as a conversation, not a one-time command.

4.

Save and Organize

When you find a prompt that works, save it! Create folders or Notion boards to store your best AI workflows and responses.

You’re not just using prompts — you’re building personalized AI systems.

Pro Tips for Maximum Results

Start with a role:

“Act as a lawyer…” / “You are now a marketing strategist…” This sets the frame for better answers.

Set the output style:

“…write in a persuasive tone, under 300 words, using storytelling.”

Stack prompts:

You can run one prompt, get a response, and build on it with new prompts. Example: Step 1: “Brainstorm 5 content ideas”

Step 2: “Turn idea #2 into a blog outline”

Step 3: “Now write the introduction for that blog”
