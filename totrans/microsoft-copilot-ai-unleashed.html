What is Prompt?

Prompts are messages written to guide AI in creating content or providing specific answers. They play a crucial role in AI's work as they define the direction and scope of the tasks it performs.

What qualities should a good prompt have?

A good prompt should possess the following qualities:

● Clear and Concise: It should be clear and concise, facilitating easy understanding for the AI to work efficiently.
● Coverage of Desired Issues: It must cover the desired issues comprehensively, enabling the AI to generate content or answers that are complete and to the point.
● Creativity: The inclusion of creative elements helps the AI in producing new and interesting content or answers.

Prompts find application in various contexts, including: ● Writing Work: Such as crafting articles, poetry, or code. ● Translation Work: Including language and document

translation.
● Research: In tasks like summarizing data and data synthesis. ● Customer Service: For answering questions and solving

problems.
Writing a good prompt is crucial as it aids the AI in working efficiently and meeting the user's needs.

Prompt Types
Prompts are categorized into four main types:

1. Objective Specification Prompt

This type of prompt is employed to outline the task's purpose for the AI, examples include "write articles about technology," "translate from Thai to English," and "write Python program code." It is essential for this prompt to clearly and concisely state its purpose, aiding the AI in understanding the intended task.

2. Information Specification Prompt

Used to communicate necessary information required by the AI, examples include "information about technology," "Thai sentences," and "Python program code." This prompt should articulate complete and accurate information, ensuring the AI operates efficiently.

3. Format Specification Prompt

This type of prompt is utilized to define the desired result's format, such as "Write an article in the form of news," "Translate language in the form of a conversation," and "Write program code in the form of a function." It is crucial for this prompt to clearly specify the format, guiding the AI in producing the intended outcome.

4. Limitations Specification Prompt

Employed to highlight various limitations in the task, examples include "Write an article no longer than 1000 words," "Translate no more than 200 words," and "Write code, and the program must run within 1 second." This prompt should explicitly state the constraints, enabling the AI to operate within the specified limitations.

Additional Prompt Categories Prompts can also be categorized based on the nature of the work:

● Work Type Specification Prompt

This type of prompt is utilized to specify the nature of the task, including "writing articles," "translating languages," "coding," etc.

● Difficulty Level Specification Prompt

Used to indicate the difficulty level of the task, examples include "write an easy article," "translate a difficult language," "write a high-level code," etc.

● Result Type Specification Prompt

This type of prompt is employed to specify the desired result type, such as "text result," "visual result," "audio result," etc.

Selecting the appropriate prompt type for the intended task enhances the efficiency of AI operations, ensuring that it meets the user's needs effectively.
