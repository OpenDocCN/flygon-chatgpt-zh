# 提示的敏感性

![图像](img/chapter_title_below.png)

ChatGPT 的响应质量和相关性受提示措辞的影响很大。然而，如果提示构造不当，这种敏感性也可能导致误导性或毫无意义的回答。

解决方案 - 仔细制定提示，指定所需格式，要求模型逐步思考，或寻求替代观点。
