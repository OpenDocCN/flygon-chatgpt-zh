## 下载

### Docker

```
docker pull apachecn0/geekdoc-chatgpt-zh
docker run -tid -p <port>:80 apachecn0/geekdoc-chatgpt-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g geekdoc-chatgpt-zh
geekdoc-chatgpt-zh <port>
# 访问 http://localhost:{port} 查看文档
```
